# FastAPI Product API Assignment

This project is a simple Product API developed using FastAPI.

## Features

1. Get all products
2. Filter products by category
3. Get in-stock products
4. Store summary
5. Search products using keyword

## API Endpoints

### Get all products
/products

### Get products by category
/products/category/{category}

### Get in-stock products
/products/instock

### Store summary
/store/summary

### Search products
/products/search/{keyword}

## Technologies Used

- Python
- FastAPI
- Uvicorn
